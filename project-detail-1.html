Sistem Perpustakaan Digital

Web Development Frontend Firebase JavaScript

Deskripsi Proyek

Sistem Perpustakaan Digital adalah platform berbasis web yang dikembangkan untuk memodernisasi sistem manajemen perpustakaan di SMA Negeri 1 Jakarta. Proyek ini dibuat sebagai bagian dari tugas mata pelajaran Teknologi Informasi dan Komunikasi (TIK) dengan tujuan menyediakan solusi digital untuk peminjaman buku, pencarian katalog, dan manajemen anggota perpustakaan.

Fitur Utama

  • Katalog buku digital dengan sistem pencarian canggih
  • Sistem peminjaman online dengan batas waktu otomatis
  • Notifikasi email untuk pengingat pengembalian
  • Dashboard admin untuk manajemen inventaris
  • Laporan statistik penggunaan perpustakaan

Teknologi yang Digunakan

Frontend

  • HTML5, CSS3, JavaScript ES6+
  • Tailwind CSS untuk styling
  • Chart.js untuk visualisasi data

Backend

  • Firebase Authentication
  • Firestore Database
  • Firebase Cloud Functions
